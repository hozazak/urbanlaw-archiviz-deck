# UrbanLaw × ArchiViz — Cinematic Deck

건축가의 시간을 되돌려주는 두 AI 제품 — **UrbanLaw**와 **ArchiViz**의 제품 발표 자료입니다.

## 🎬 발표자료 보기

👉 **[Live Demo (GitHub Pages)](https://hozazak.github.io/urbanlaw-archiviz-deck/)**

## 📦 구성

- 단일 HTML 파일 (1920×1080 시네마틱 슬라이드 23p)
- Scroll-snap 기반 키노트형 deck
- Pretendard / Satoshi 웹폰트
- 키보드 네비게이션 (방향키 / Space / Home·End / 숫자키)

## 🏛️ 두 제품

### UrbanLaw — 건축법규 AI 컨설턴트
주소 하나로 토지정보·지구단위계획·건축물대장을 모아, 법령 + 판례 + 조례를 근거로 전문가 분석을 내놓는 FastAPI 기반 SPA.

### ArchiViz — 건축 전용 AI 스튜디오
건축가의 실무 언어로 설계된 AI 비주얼라이제이션 스튜디오. 프롬프트 대신 태그 선택으로, 참조 이미지 기반으로 의도를 보존.
